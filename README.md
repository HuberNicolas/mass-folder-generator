# mass-folder-generator

> An Excel VBA macro that automatically creates bulk folder structures on the local file system from a spreadsheet list.

## ✨ TL;DR
Open the Excel file and run the macro (`Alt+F8` → `GenerateFolders`).

## 🚀 Features
- **Bulk folder creation:** Creates any number of folders based on an Excel list.
- **No external dependencies:** Runs entirely within Excel via VBA.
- **Configurable:** Folder names and target paths set directly in the spreadsheet.

---

## 🛠️ Installation

### Prerequisites
- Microsoft Excel (with macros enabled)

### Steps
1. Clone the repository:
```bash
git clone https://github.com/HuberNicolas/mass-folder-generator
cd mass-folder-generator
```
2. Open `Mass Folder Generator.xlsm` in Excel.
3. Enable macros if prompted.

---

## 💻 Usage

1. Enter folder names in the Excel spreadsheet.
2. Configure the target path.
3. Run the macro: `Alt+F8` → select macro → `Run`.
